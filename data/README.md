# Dataset Archive

本目录用于沉淀翻译相关回归样本。

## 目录约定

- `20260315/`
  - 第一批样衣/工艺单翻译样本
  - 同时包含原文 PDF 与人工翻译 PDF
- `20260324/`
  - 第二批工艺单/TP/多版式样本
  - 同时包含原文 PDF、部分人工翻译 PDF、部分人工翻译 Excel
- `feedback-translation/`
  - 旧的案例化开发样本
- `test/`
  - 开发期通用测试文件

## 文件角色

- `source_pdf`
  - 待抽取、待翻译的原始文档
- `reference_pdf`
  - 人工翻译或业务确认过的 PDF 版本
- `reference_xlsx`
  - 人工翻译或整理后的结构化参考答案

## 当前归档方式

当前不移动原文件，避免打断已有脚本和人工使用路径。

每个日期目录使用 `manifest.json` 记录：

- 样本分组
- 原文/参考文档配对关系
- 版式类型与用途说明
- 是否适合用于抽取/翻译回归

## 使用建议

1. 优先用 `manifest.json` 里的样本组而不是手工猜配对关系。
2. 抽取算法回归时，至少同时覆盖：
   - `sketch/comment`
   - `tp/table-heavy`
   - `mixed/reference`
3. 不要只用单一文档做规则验证，避免过拟合。
