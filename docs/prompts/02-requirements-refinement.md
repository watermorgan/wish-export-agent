# Prompt 02: 需求细化

```text
请基于当前项目文档，继续细化外贸助手 V1 需求。

先阅读：
- `docs/product/01-prd.md`
- `docs/product/03-workflows.md`
- `docs/product/04-skills-and-orchestration.md`

要求：
- 只围绕 V1 范围展开，不引入自动报价、自动发外部消息、自动写 ERP/CRM
- 只服务业务员和主管
- 保持“技能卡片 + 人工触发组合 + 关键步骤人工确认”的产品原则
- 输出必须服务于后续页面设计和开发拆解

请完成以下内容：
1. 把现有需求补成标准化功能需求列表
2. 按“用户目标 -> 页面动作 -> 系统行为 -> 输出结果 -> 人工确认点”整理
3. 补充异常场景和失败回退
4. 标出哪些是 V1 必做，哪些是 V1.1 可延后

输出格式：
- 先给高优先级需求清单
- 再给每条需求的验收标准
- 最后给风险和待确认问题
```
