# Memory

Use this folder for accepted, reusable memory that other humans or AI agents
should be able to rely on.

## Layers

- Project memory: durable product, architecture, contract, or requirement facts.
- Control-plane memory: reusable routing, handoff, evidence, or agent operating
  rules.
- Current-agent memory: lessons from current execution that have been reviewed
  and are useful beyond one task.

Raw process artifacts belong under `.ai-control/raw/`. Research or findings that
may become memory should enter a review queue first, then be promoted here only
as compact accepted conclusions.
